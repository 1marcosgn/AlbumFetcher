# AlbumFetcher

This app has been created:

- Using Auto Layout
- No Storyboards or Nibs
- No Third Party Libraries

## Goal.

Displays the top 100 albums across all genres using Apple’s RSS generator found here: https://rss.itunes.apple.com/en-us.
